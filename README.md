# ADN CONVERTER V7.0 - Psychonarrative Historical

ADN CONVERTER V7.0 est une plateforme de traitement d'image algorithmique de pointe, conçue pour réinterpréter des photographies à travers les prismes stylistiques et philosophiques des plus grands maîtres de la peinture.

Le système utilise une architecture cognitive en 6 dimensions, permettant une fidélité artistique et historique sans précédent.

## Moteur Artistique Universel V7.0

Le système suit un pipeline de traitement en 6 phases inspiré des techniques de conservation et de restauration d'art :

### Phase 1 : Temps-Espace (Temporalité)
Ajuste la perception temporelle et la dynamique spatiale de l'image.

| Sous-paramètre | Description | Effet |
|----------------|-------------|-------|
| **Éternité Figée** (Timeless Frozen) | Suspension temporelle, calme absolu | Focus doux, luminosité subtile, surface lisse |
| **Durée Atmosphérique** (Atmospheric Duration) | Passage du temps, brume temporelle | Grain subtil, focus atmosphérique, dégradé de luminosité |

### Phase 2 : Matière-Esprit (Matérialité)
Crée une texture de surface unique évoquant soit la transcendance soit la tangibilité.

| Sous-paramètre | Description | Effet |
|----------------|-------------|-------|
| **Matière comme Expression** (Material as Expression) | Texture brute et tangible | Grain texturé, contraste défini, micro-détails |
| **Divin à travers la Matière** (Divine through Material) | Transcendance spirituelle | Focus doux, luminosité céleste, effet de rayonnement |

### Phase 3 : Lumière-Révélation (Luminosité)
Gère la puissance lumineuse et la philosophie de l'éclairage.

| Sous-paramètre | Description | Effet |
|----------------|-------------|-------|
| **Lumière comme Preuve Divine** (Light as Divine Proof) | Illumination transcendante | Luminosité élevée, halos subtils, dégradés doux |
| **Lumière comme Outil Dramatique** (Light as Dramatic Tool) | Clair-obscur théâtral | Contraste puissant, vignettage prononcé, ombres profondes |

### Phase 4 : Psychologie du Regard (Orchestration)
Orchestre le regard du spectateur via des techniques émotionnelles et analytiques.

| Sous-paramètre | Description | Effet |
|----------------|-------------|-------|
| **Immersion Émotionnelle** (Emotional Immersion) | Absorption totale du spectateur | Contraste modéré, grain intime, bords adoucis |
| **Examen Microscopique** (Microscopic Scrutiny) | Précision analytique | Netteté maximale, micro-contraste, clarté des bords |
| **Orchestration Dramatique** (Dramatic Orchestration) | Composition théâtrale | Contraste élevé, netteté sélective, profondeur simulée |

### Phase 5 : Contrainte Historique (Authenticité)
Applique les contraintes chromatiques et stylistiques d'une période artistique.

| Période | Époque | Caractéristiques |
|---------|--------|------------------|
| **Âge d'Or** (Golden Age) | 17ème siècle | Vermeer, lumière douce, intérieurs domestiques |
| **Contemporain** (Contemporary) | 21ème siècle | Neutre, préservation des couleurs originales |
| **Moderne** (Modern) | 20ème siècle | Bacon, Chagall, expressionnisme |
| **Impressionnisme** (Mid-20th) | 19ème siècle | Klimt, motifs ornementaux |
| **Baroque** | 16ème-17ème siècle | Caravaggio, clair-obscur dramatique |
| **Haute Renaissance** (High Renaissance) | 15ème-16ème siècle | Turner, Friedrich, atmosphérique |
| **Début Renaissance** (Early Renaissance) | 14ème-15ème siècle | Perspectives, symétrie |

### Phase 6 : Finition "Master"
Ajoute un Lustre de Maître final pour une présence vibrante et professionnelle.

## Fonctionnalités Clés

- **Téléchargement Précis** : Export de l'image originale traitée sans bordure de canvas
- **Interface en Français** : Tous les paramètres et sous-paramètres traduits
- **Curseurs Précis** : Ajustement fin de chaque effet avec curseurs 0-100%
- **Import de Paramètres** : Chargement rapide des paramètres optimaux par maître
- **Détection Automatique** : Suggestion intelligente du style optimal pour chaque image
- **Rendu Temps Réel** : Prévisualisation instantanée des modifications

## Stack Technique

- **Frontend** : React, Vite, Tailwind CSS, Shadcn UI
- **Rendu** : React-Konva (canvas haute performance)
- **Backend** : Node.js, Express, PostgreSQL, Drizzle ORM
- **Traitement** : Algorithmes de traitement d'image en JavaScript pur

## Installation

```bash
npm install
npm run db:push
npm run dev
```

## Utilisation

1. **Importer une image** via le bouton "Import Image"
2. **Sélectionner un maître** ou utiliser la détection automatique
3. **Ajuster les paramètres** avec les curseurs pour chaque dimension
4. **Télécharger** l'image finale avec le bouton "Télécharger"
5. **Importer des paramètres** d'autres maîtres via "Importer Paramètres"
